## Задание 1

Необходимо реализовать методы для списка.

Так как номер варианта — **30**, а он является чётным, в работе используется: односвязный список

Каждому студенту необходимо реализовать 6 методов.

Формулы для определения номеров заданий:

1. (v + 7) % n + 1
2. (v + 14) % n + 1
3. (v + 21) % n + 1
4. (v + 28) % n + 1
5. (v + 35) % n + 1
6. (v + 42) % n + 1

где:

v = 30
n = 30

Расчёт номеров заданий:

1. (30 + 7) % 30 + 1 = 8
2. (30 + 14) % 30 + 1 = 15
3. (30 + 21) % 30 + 1 = 22
4. (30 + 28) % 30 + 1 = 29
5. (30 + 35) % 30 + 1 = 6
6. (30 + 42) % 30 + 1 = 13

Следовательно, для варианта 30 необходимо реализовать следующие методы:

Метод 8
void insert_before(List list, Node node, int value)

Вставка нового элемента перед заданным элементом списка.

Метод 15
int get_length(List list)

Подсчёт длины списка.

Метод 22
void reverse(List list)

Разворот списка.

Пример: [1, 2, 3, 4, 5] -> [5, 4, 3, 2, 1]

Метод 29
int get_min(List list)

Возвращает значение минимального элемента списка.

Если список пустой, должна быть обработана ошибка.

Метод 6
void delete_after(List list, Node node)

Удаление элемента после заданного элемента.

Метод 13
Node find(List list, int value)

Поиск элемента списка по значению.

## Задание 4

Смоделировать ситуацию обслуживания клиентов в банке.

Условия:

клиенты случайным образом помещаются в одну из M очередей обслуживания;

случайным образом выбирается одна из очередей;

клиент из выбранной очереди обслуживается и выбывает из неё.

При каждой операции необходимо указывать:

номер добавленного клиента;

номер обслуженного клиента;

состояние очередей.
