# Лабораторная работа № 2

## Цель работы

Изучить вопросы применения основных методов построения алгоритмов:

- метода «разделяй и властвуй»;
- динамического программирования.

## Ключевые понятия

- основные методы построения рекурсивных алгоритмов;
- метод «разделяй и властвуй»;
- динамическое программирование;
- нисходящий метод динамического программирования;
- восходящий метод динамического программирования;
- принцип оптимальности Беллмана.

## Задания для самостоятельной работы

В данной лабораторной работе выбраны следующие задания:

1. Задание №1;
2. Задание №2а;
3. Задание №3;
4. Задание №4.

В каждом задании, кроме написания программного кода, необходимо дать оценку скорости роста времени работы алгоритма с использованием O-символики.

## Задание 1

**Сравнение способов вычисления 35-го числа Фибоначчи.**

Необходимо сравнить время вычисления 35-го числа Фибоначчи при помощи следующих методов:

1. формула Бине;
2. итерационная формула;
3. метод «разделяй и властвуй»;
4. метод нисходящего динамического программирования;
5. метод восходящего динамического программирования.

Дополнительно:

- найти точное значение 175-го числа Фибоначчи за `O(log N)`.

---

## Задание 2а

**Нахождение максимального значения в массиве методом «разделяй и властвуй».**

Необходимо описать алгоритм нахождения максимального значения в массиве, используя принцип «разделяй и властвуй».

Идея алгоритма:

1. Разделить массив на две части.
2. Рекурсивно найти максимум в левой части.
3. Рекурсивно найти максимум в правой части.
4. Сравнить два найденных значения.
5. Вернуть большее значение как максимум всего массива.

Также требуется указать оценку временной сложности алгоритма.

---

## Задание 3

**Задача о Ханойской башне.**

Необходимо решить задачу о Ханойской башне.

Суть задачи:

- есть несколько дисков разного размера;
- изначально все диски находятся на одном стержне;
- нужно перенести все диски на другой стержень;
- за один ход можно переносить только один диск;
- нельзя класть больший диск на меньший.

Также требуется указать оценку временной сложности алгоритма.

---

## Задание 4

**Задача о мячике на лесенке.**

На вершине лесенки, содержащей `N` ступенек, находится мячик.  
Мячик начинает прыгать вниз, к основанию.

За один прыжок мячик может переместиться:

- на следующую ступеньку;
- на ступеньку через одну;
- на ступеньку через две.

Например, если мячик находится на 8-й ступеньке, то он может переместиться на: 7-ю, 6-ю или 5-ю ступеньку
Необходимо написать программу, которая определяет количество всевозможных маршрутов мячика с вершины на землю.

Ограничения: 1 <= N <= 30
