# Лабораторная работа № 4

## Задания 3–6

---

## Задание 3

Написать рекурсивную процедуру построения идеально сбалансированного дерева для произвольных `N` узлов, хранящихся в файле.

Для полученного дерева необходимо:

1. определить высоту дерева;
2. определить число уровней;
3. определить, сколько путей длины 3 существует в этом дереве;
4. для каждого пути длины 3 перечислить узлы, входящие в путь;
5. построить расширенное дерево;
6. найти длину внешнего пути расширенного дерева;
7. найти длину внутреннего пути расширенного дерева.

---

## Задание 4

Написать процедуру вычисления арифметического выражения в дереве, используя постфиксный обход.

Постфиксный обход выполняется по правилу: левое поддерево → правое поддерево → корень

---

## Задание 5

Написать рекурсивную процедуру подсчёта листьев в дереве из N узлов.

Листом считается узел, у которого отсутствуют потомки.

---

## Задание 6

Написать процедуры нерекурсивных обходов бинарного дерева:

префиксного обхода;
инфиксного обхода;
постфиксного обхода;
поуровневого обхода.

Для поуровневого обхода рекомендуется использовать FIFO-очередь.

Для остальных обходов рекомендуется использовать стек.

Виды обходов
Префиксный обход
корень → левое поддерево → правое поддерево
Инфиксный обход
левое поддерево → корень → правое поддерево
Постфиксный обход
левое поддерево → правое поддерево → корень
Поуровневый обход
слева направо, сверху вниз
